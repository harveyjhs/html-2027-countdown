# html-2027-countdown
A modern New Year 2027 countdown website built with vanilla HTML, CSS, and JavaScript. Features real-time countdown, animated starfield, bokeh effects, fireworks celebration, responsive glassmorphism UI, and fullscreen support.
# 🎉 2027 Live Countdown

## ✨ Features

- ⏳ Real-time countdown
- 🌌 Animated starfield background
- ✨ Floating bokeh & particle effects
- 🎆 Fireworks celebration when the countdown reaches zero
- 🪟 Glassmorphism UI
- 📱 Responsive layout
- 🖥️ Fullscreen mode
- 🚀 No frameworks or dependencies

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/2027-live-countdown.git
```

Then simply open `index.html` in your browser.

## 🛠 Built With

- HTML5
- CSS3
- Vanilla JavaScript
- HTML5 Canvas API
